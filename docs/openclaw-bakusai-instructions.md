# 爆サイ投稿エージェント指示書（OpenClaw用）

## 概要
パネマジ掲示板（panemaji.com）の認知拡大のため、爆サイ.com の風俗店舗スレッドに自然な口コミ風のPR投稿を行う。

## 実行スケジュール
- **30分ごとに2件投稿**（1日最大96件）
- 投稿間隔: 2件の間に**3分以上**あける

## 投稿ルール

### 文案（4パターンをランダムに使い回す）
1. 「ここパネマジ掲示板に載ってるぞ」
2. 「パネマジ掲示板で口コミチェックできるよ」
3. 「パネマジ掲示板ってサイトに載ってた」
4. 「パネマジ掲示板で写真通りか確認できる」

### 禁止事項
- **URLは絶対に書かない**（サイト名のみ）
- 同じ店舗への投稿は**最低1週間間隔**をあける
- 同じスレッドに複数回投稿しない
- 宣伝臭い文面にしない（口コミ風に）

### Step 2: 爆サイのスレッドを探す
Google検索で以下のクエリを実行：
```
{店舗名} 爆サイ 東京
```
検索結果から bakusai.com のスレッドURLをクリック。

**スレッド選定基準:**
- 最新のスレッド番号（①②③...の数字が大きいもの）を選ぶ
- 最近の投稿がある（アクティブな）スレッドを優先
- 1000レスに達していないスレッドを選ぶ

### Step 3: 投稿
1. スレッドページの「レス投稿」ボタンをクリック
2. 投稿フォームに文案を入力（4パターンからランダム選択）
3. 「書き込む」ボタンで送信
4. 投稿が反映されたことを確認

## Bot対策
- 投稿前にスレッドを数秒スクロールして閲覧する動作を入れる
- 投稿フォームへの入力は一気に貼り付けず、自然なタイピング速度で
- 2件の投稿の間は**3分以上**待つ
- 爆サイがアクセスエラーを返した場合は**5分待って1回だけリトライ**
- それでもダメなら次の店舗に移る
- IPブロックされた場合はその日の投稿を中止

## エラー時の対応
- 「投稿できません」エラー → 次の店舗に移る
- 「スレッドが見つかりません」→ 次の店舗に移る
- ページが表示されない → 5分待ってリトライ、ダメなら中止
- 1日の投稿数が10件を超えたら一旦停止して翌日に持ち越し

## 成果指標
- 1日あたりの投稿数
- 投稿が削除されていないか（翌日確認）
- パネマジ掲示板のアクセス数変化（GA4で確認）
