# パネマジ 独自MECEエリア定義

**このドキュメントが唯一の正**。外部ソース（シティヘブン等）のエリア区分はここにマップする。新規追加・統合は両方ここと `scripts/migrate-areas-v4.mjs` の `UNIFIED_AREAS` を同期させること。

## 原則

1. **ME (Mutually Exclusive)** — 同一エリアは1つのスロットにのみ割り当て。「新宿・歌舞伎町」と「新宿・大久保・高田馬場・中野」のような重複を作らない
2. **CE (Collectively Exhaustive)** — 全都道府県に「〇〇その他」枠（`display_order = 99`）を用意し、マッピング漏れを拾う
3. **0店舗のエリアは作らない / 残さない** — migrateは空エリアを削除する

## 運用ルール

1. 新しいスクレイプソースを追加する際、エリアマッピングは必ず UNIFIED_AREAS.keywords で拾えるようにキーワード追加で対応する
2. スクレイパーは完了後に必ず `scripts/migrate-areas-v4.mjs` を呼ぶ（pipeline.mjs に組み込み済み）
3. 新しい独自エリアを切る時は:
   - この `docs/area-definition.md` を更新
   - `scripts/migrate-areas-v4.mjs` の UNIFIED_AREAS も更新
   - 両者を1コミットで揃える
